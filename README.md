# Kaspi Bot

Telegram-бот для продавцов на Kaspi. Уведомления о заказах, мониторинг цен конкурентов, контроль остатков.

---

## Запуск за 5 минут (демо-режим)

### 1. Получи токен Telegram бота
- Открой @BotFather в Telegram
- Напиши `/newbot`
- Придумай имя → получи токен вида `1234567890:AAH...`

### 2. Создай .env файл
```
BOT_TOKEN=твой_токен_сюда
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти
```bash
python bot.py
```

### 5. Открой бота в Telegram
- `/register` → введи любое слово как токен → введи название магазина
- Жди 2 минуты → придёт тестовый заказ
- Жди 5 минут → придёт уведомление о цене конкурента

---

## Переключение на реальный режим (токен от сестры)

### 1. В config.py измени:
```python
DEMO_MODE = False
```

### 2. Запусти бот, напиши /register
Теперь бот запросит настоящий токен Kaspi.

### Как получить токен Kaspi:
1. kaspi.kz/merchants
2. Настройки → API интеграция
3. Сгенерировать токен → скопировать

## Структура проекта

| Файл | Что делает |
|------|-----------|
| bot.py | Telegram бот, все команды |
| kaspi_api.py | Получение заказов и товаров через Kaspi API |
| parser.py | Парсинг цен конкурентов со страниц Kaspi |
| scheduler.py | Фоновые проверки по расписанию |
| storage.py | Сохранение данных в JSON файлы (без БД) |
| mock_data.py | Фейковые данные для демо |
| config.py | Настройки |

---

## Что делает бот

**Каждые 2 минуты** — проверяет новые заказы. Если пришёл новый → уведомление в Telegram.

**Каждые 30 минут** — обходит страницы всех товаров на Kaspi, собирает цены всех продавцов этого товара, если конкурент стал дешевле → уведомление.

**Каждый час** — проверяет остатки. Если товар заканчивается (меньше 5 штук) или закончился → уведомление.

---

## Если парсер цен перестал работать

Kaspi иногда меняет HTML. Открой страницу любого товара в браузере, DevTools → Elements, найди блок с продавцами и обнови классы в parser.py в функции `_parse_sellers`.
